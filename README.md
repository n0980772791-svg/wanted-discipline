# 紀律懸賞追蹤 PWA

## 安裝到手機方法

### 方法一：GitHub Pages（免費，推薦）

1. 到 https://github.com 註冊/登入
2. 點右上角「+」→「New repository」
3. Repository name 填：`wanted-discipline`
4. 選 **Public**，按「Create repository」
5. 點「uploading an existing file」
6. 把這個資料夾裡的所有檔案拖進去上傳：
   - index.html
   - manifest.json
   - sw.js
   - icon-192.svg
   - icon-512.svg
7. 按「Commit changes」
8. 到「Settings」→「Pages」→ Source 選「main branch」→ Save
9. 等 1-2 分鐘，網址會是：`https://你的帳號.github.io/wanted-discipline`

### 加到手機主畫面

**iPhone (Safari)：**
1. 用 Safari 打開網址
2. 點底部分享按鈕 □↑
3. 選「加入主畫面」
4. 點「新增」

**Android (Chrome)：**
1. 用 Chrome 打開網址
2. 點右上角三個點 ⋮
3. 選「新增到主畫面」或「安裝應用程式」

## 完成！
打開就像真正的 App，全螢幕、可離線使用、資料存在手機本地。
